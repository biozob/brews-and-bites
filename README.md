# BREWS AND BITES

### This is a presentation project.

## Run Locally

Clone the project

```bash
  git clone https://github.com/biozob/brews-and-bites.git
```

Go to the project directory

```bash
  cd brews-and-bites
```

Run with Docker

```bash
  docker-compose up
```
